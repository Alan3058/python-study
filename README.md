# why study python
偶尔在少年宫图书馆随意翻阅了一本python书籍，报着好奇心翻了几页。它简单的语法吸引到了我，用它来写些简单的工具应该是个不错的选择。同一个功能，相比java语言实现的代码量，完全不是一个级别，真心不错，强力推荐。  
它是一种脚本语言，和javascript、gradle、shell有很多共同之处。上手快速，使用自然，不繁杂。
